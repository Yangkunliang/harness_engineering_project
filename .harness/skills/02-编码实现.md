## 6步法

### 步骤1：编写接口定义

1. 创建 Controller 类
2. 定义 REST 接口
3. 编写请求/响应 DTO

### 步骤2：编写数据模型

1. 创建 Entity 实体类
2. 定义数据库映射关系
3. 创建 Repository 接口

### 步骤3：实现业务逻辑

1. 创建 Service 接口
2. 实现业务逻辑
3. 处理业务异常

### 步骤4：添加异常处理

1. 使用统一异常处理
2. 定义业务错误码
3. 记录日志

### 步骤5：编写单元测试

1. 编写测试用例
2. 覆盖核心逻辑
3. 验证边界条件

### 步骤6：代码自检

## 自检清单

- [ ] 代码符合阿里巴巴Java开发手册
